# Notebooks Directory

This directory contains Jupyter notebooks for exploratory data analysis (EDA) and reporting.

## Notebook Files

- **01_data_exploration.ipynb** - Initial data exploration and summary statistics
- **02_regional_analysis.ipynb** - Detailed regional performance analysis
- **03_trend_analysis.ipynb** - Time-series and trend analysis
- **04_visualization.ipynb** - Data visualization and insights

## How to Run

```bash
jupyter notebook
```
