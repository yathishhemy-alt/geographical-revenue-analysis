# Geographical Revenue AnalysisThis project analyzes revenue trends across different geographical regions using Python, Pandas, Excel, and Power BI. The analysis helps identify top-performing regions, sales patterns, and business insights through data visualization and dashboards.
# Geographical Revenue Analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Excel

## Project Objective
Analyze regional revenue trends and identify top-performing geographical areas.
