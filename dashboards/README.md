# Dashboards Directory

This directory contains Power BI dashboards and visualization files.

## Files

- **sales_dashboard.pbix** - Main sales performance dashboard
- **regional_analysis.pbix** - Regional breakdown dashboard
- **trend_dashboard.pbix** - Revenue trend and forecasting dashboard

## How to Use

1. Open the .pbix files with Power BI Desktop
2. Connect to your data sources
3. Refresh the data
4. View and share the dashboards
